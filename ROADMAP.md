# ROADMAP

AS Code is evolving into a fast, lightweight, general-purpose local AI runtime. Our focus is on stability and performance for "real-world" hardware.

## ✅ Completed / Advanced

* Core Modular Architecture
* LiteRT-LM Windows Runtime
* GPU Acceleration & VRAM-Aware Loading
* SSE Streaming Support
* OpenAI-Compatible API (v1)
* Smart Routing Base
* Premium Minimal Web UI
* Model Hot-Swap Support
* **Multi-Role Support:** Dedicated roles for Chat, Code, and Reasoning with Gemma.

## 🚧 Current Priorities

* **Runtime Stability:** Hardening the LiteRT-LM execution layer.
* **Lightweight Optimization:** Reducing idle footprint and peak resource usage.
* **Installation Experience:** Improving one-click setup and dependency management.
* **Browser UX:** Enhancing the local chat interface for productivity and creative tasks.
* **OpenAI Compatibility:** Ensuring 100% parity for standard tools.
* **Hardware Adaptability:** Better auto-tuning for different GPU/RAM combinations.

## 🔮 Future Priorities

* **Smarter Routing:** Intelligent model selection based on task complexity.
* **Productivity Workflows:** Built-in tools for common business and writing tasks.
* **Local Agents:** Lightweight autonomous systems for local file/task management.
* **Better Conversational Behavior:** Fine-tuned local instruction following.
* **Advanced Memory Optimization:** Context compression and advanced quantization support.
* **Automatic Model Management:** Streamlined downloading and versioning (HuggingFace integration).
* **Optional VSCode Tooling:** Deep integration for developers who want a local-first alternative to Copilot.
